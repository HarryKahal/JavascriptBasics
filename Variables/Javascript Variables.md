Javascript Variables

Definition :
  Variables are container which used to store data.

Ways to declare a variable :

  1. Automatically : 
      x = 5;
  2. Using var keyword : 
      var x = 5;
  3. Using let keyword : 
      let x = 5;
  4. Using const keyword : 
      const x = 5;
